# Обнаружение изменений на спутниковых снимках

Проект реализует классический пиксельный алгоритм обнаружения изменений на парах RGB-снимков LEVIR-CD. Основной результат работы алгоритма - бинарная маска изменений, которая сравнивается с эталонной маской по `Precision`, `Recall` и `F1-score`.

## Быстрый запуск

```powershell
python main.py --experiment comparison --no_plot
```

## Основные файлы

- `config.yaml` - фиксированная конфигурация алгоритма после подбора параметров.
- `main.py` - единая точка запуска экспериментов.
- `pipelines/adaptive_algorithm.py` - основной предложенный алгоритм.
- `pipelines/baseline_methods.py` - базовые методы для сравнения.
- `experiments/run_comparison.py` - сравнительный эксперимент.
- `analysis/ablation_study.py` - оценка вклада адаптивных улучшений.
- `analysis/monte_carlo_optimization.py` - подбор параметров методом Monte Carlo.
- `scripts/generate_visual_report.py` - генерация визуального отчета по нескольким примерам.
- `results/` - сохраненные таблицы результатов экспериментов.

## Логика алгоритма

1. Загружаются два снимка одной территории: `T1` и `T2`.
2. При необходимости выполняется предобработка: сглаживание, CLAHE, радиометрическая нормализация.
3. Строится карта различий: комбинированная разность, CVA-компонента или MADI.
4. При включенном многоуровневом режиме карта различий считается на Gaussian pyramid.
5. Карта корректируется handcrafted-приорами: стабильная растительность `Excess Green`, контуры, тени и признаки застройки.
6. Score-map переводится в бинарную маску выбранным методом пороговой обработки.
7. Маска очищается морфологией, фильтрацией связных компонент и building-support фильтром.
8. Финальная маска сравнивается с эталоном по `Precision`, `Recall` и `F1-score`.
